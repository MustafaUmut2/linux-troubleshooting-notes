### Title
No Internet Access on Windows Laptop

### Description
User reports Wi-Fi is connected, but there is no internet access on a Windows laptop.

### Environment
- OS: Windows 10
- Network: Home Wi-Fi
- Other devices: Working normally

### Troubleshooting Steps
- Verified other devices had internet access
- Restarted Wi-Fi adapter (no change)
- Disabled VPN (no change)
- Ran Windows network troubleshooter
- Reset network settings
- Checked IP and DNS settings (automatic)
- Cleared TCP/IP stack and DNS cache
- Updated Wi-Fi adapter driver

### Resolution
Uninstalled the Wi-Fi adapter driver via Device Manager and allowed Windows to reinstall it automatically after reboot.

### Status
Closed
